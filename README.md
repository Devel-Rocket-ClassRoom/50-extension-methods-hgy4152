# 확장 메서드 만들기

## 사용 방법

1. Visual Studio에서 `Chapter50-ExtensionMethods.slnx`를 엽니다
2. 솔루션 탐색기에서 프로젝트를 선택하고 `README.md`를 읽습니다
3. `Program.cs`에 코드를 작성합니다
4. 해당 프로젝트를 시작 프로젝트로 설정하고 실행합니다 (`Ctrl+F5`)

## 코딩 연습

`CodingPractice` 프로젝트의 README를 따라 교안 예제 코드를 직접 작성하고 실행합니다.

| 프로젝트 | 내용 |
|----------|------|
| CodingPractice-01 | 교안 코드 따라치기 - 확장 메서드 기본 (3개 과제) |
| CodingPractice-02 | 교안 코드 따라치기 - 타입 확장 (2개 과제) |
| CodingPractice-03 | 교안 코드 따라치기 - 확장 메서드 활용 (3개 과제) |
| CodingPractice-04 | 교안 코드 따라치기 - 제약 사항 (3개 과제) |

## 연습문제

각 프로젝트의 README에 있는 문제를 읽고 코드를 작성합니다.

| 프로젝트 | 문제 |
|----------|------|
| Chunk | 컬렉션 청킹 확장 메서드 |
| Palindrome | 회문 판별 확장 메서드 |
| Shuffle | 컬렉션 셔플 확장 메서드 |
